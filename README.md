# Computer Vision with OpenCV

A collection of practical computer vision applications built using Python and OpenCV.  
These projects demonstrate real-time video processing, image manipulation, and object detection techniques.

## Projects Included

1. Live Camera Streaming
2. Grayscale & Inverted Frame Processing
3. Real-time Face Detection using Haar Cascade
4. Image Creation with Text and Shape Rendering
5. Real-time Color Detection using HSV Color Space

## Technologies Used
- Python
- OpenCV
- NumPy
